🔐 Email + Password Aman
📩 Cara masuk pakai email 1. Daftar pakai email aktif. 2. Buka email dari Supabase Auth. 3. Klik Confirm your mail. 4. Kembali ke web ini, lalu pilih Login. Kalau email belum masuk, cek folder Spam/Promotions atau tunggu 1–2 menit. Jika tombol daftar diam/stuck: buka Supabase → Authentication → Providers → Email, pastikan Email provider ON dan Allow signups ON.
Catatan: agar akun dan data bisa dipakai di HP maupun laptop, aktifkan Firebase Cloud di kode bagian FIREBASE_CONFIG. Jika belum diisi, web tetap berjalan lokal di perangkat ini.
